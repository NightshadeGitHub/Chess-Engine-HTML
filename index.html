Chess Project

A web-based chess game

About

Welcome to Chess Project, a web-based chess game where you can play chess against the computer or with friends online.

This project is a web-based chess game that allows users to play chess online. It features:

  • Play against the computer with adjustable difficulty levels.
  • Play with friends in real-time.
  • Analyze games with built-in tools.
